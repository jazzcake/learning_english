# Chapter 03

이번 챕터는 **동명사(Gerund)**입니다. 동사원형에 `~ing`를 붙여서 명사처럼 쓰는 문법입니다. 특히 **"어떤 동사 뒤에는 무조건 동명사만 와야 하는지"**가 중학교 내신 시험의 단골 문제입니다.

---

### Chapter 03. 동명사 (주어/목적어 역할과 관용 표현)

### 0. 핵심 요약 (Key Points)

- **형태:** 동사원형 + ing (예: eat → eating)
- **의미:** ~하는 것, ~하기
- **주요 역할:**
    1. **주어:** 문장 맨 앞 (단수 취급)
    2. **목적어:** `enjoy`, `finish`, `stop`, `mind` 등의 동사 뒤
    3. **전치사의 목적어:** `in`, `at`, `of`, `about` 등 전치사 뒤에는 무조건 `ing`!

---

### 1. 예문 연습 (Sentence Practice)

1. 동전을 수집하는 것은 / 이다 / 나의 취미 (주어)
    
    **Collecting coins** / is / my hobby.
    
2. 나는 / 즐긴다 / 축구하는 것을 / 방과 후에 (목적어)
    
    I / enjoy / **playing soccer** / after school.
    
3. 그는 / 멈췄다 / 담배 피우는 것을 / 작년에 (목적어)
    
    He / stopped / **smoking** / last year.
    
4. 너는 / 끝냈니 / 너의 방 청소하는 것을? (목적어)
    
    Have you / finished / **cleaning your room**?
    
5. 그녀는 / 능숙하다 / 수영하는 것에 (전치사의 목적어)
    
    She / is good at / **swimming**.
    
6. 일기를 쓰는 것은 / 이다 / 좋은 습관 (주어)
    
    **Keeping a diary** / is / a good habit.
    
7. 꺼리시나요? (괜찮으신가요?) / 창문을 여는 것을 (목적어)
    
    Do you / mind / **opening the window**?
    
8. 우리는 / 포기했다 / 그 산을 오르는 것을 (목적어)
    
    We / gave up / **climbing the mountain**.
    
9. 고마워 / 나를 초대해 줘서 / 저녁 식사에 (전치사의 목적어)
    
    Thank you for / **inviting me** / to dinner.
    
10. 어때? / 영화 보러 가는 게 (관용 표현)
    
    How about / **going to the movies**?
    
11. 피하는 게 좋다 / 정크 푸드를 먹는 것을 (목적어)
    
    You should avoid / **eating junk food**.
    
12. 톰은 / 관심이 있다 / 사진을 찍는 것에 (전치사의 목적어)
    
    Tom / is interested in / **taking photos**.
    
13. 채소를 먹는 것은 / 중요하다 / 건강에 (주어)
    
    **Eating vegetables** / is important / for health.
    
14. 그녀는 / 계속했다 / 질문하는 것을 (목적어)
    
    She / kept / **asking questions**.
    
15. 나는 / 연습했다 / 피아노 치는 것을 / 한 시간 동안 (목적어)
    
    I / practiced / **playing the piano** / for an hour.
    
16. 자전거를 타는 것은 / 이다 / 아주 재미있는 (주어)
    
    **Riding a bike** / is / a lot of fun.
    
17. 그는 / 나갔다 / 작별 인사를 하지 않고 (전치사의 목적어)
    
    He / went out / without / **saying goodbye**.
    
18. 나는 / 학수고대하고 있다 / 너를 보는 것을 (관용 표현 - 중요!)
    
    I / look forward to / **seeing you**.
    
19. 미루지 마라 / 너의 숙제 하는 것을 / 내일로 (목적어)
    
    Don't put off / **doing your homework** / until tomorrow.
    
20. 제안합니다 / 택시를 타는 것을 (목적어)
    
    I / suggest / **taking a taxi**.
    
21. 나는 / 지겹다 (지쳤다) / 그를 기다리는 것이 (전치사의 목적어)
    
    I / am tired of / **waiting for him**.
    
22. 그들은 / 부인했다 / 그 꽃병을 깬 것을 (목적어)
    
    They / denied / **breaking the vase**.
    
23. 실수하는 것은 / 이다 / 배움의 일부 (주어)
    
    **Making mistakes** / is / part of learning.
    
24. 상상해 봐라 / 하늘을 나는 것을 (목적어)
    
    Imagine / **flying in the sky**.
    
25. 그는 / 두려워한다 / 대중 앞에서 연설하는 것을 (전치사의 목적어)
    
    He / is afraid of / **making a speech** / in public.
    
26. 나는 / 싶다 (기분이 든다) / 피자를 먹고 (관용 표현)
    
    I / feel like / **eating pizza**.
    
27. 게임을 하는 것 대신에 / 책을 좀 읽어라 (전치사의 목적어)
    
    Instead of / **playing games**, / read some books.
    
28. 그녀는 / 그만뒀다 / 그 회사에서 일하는 것을 (목적어)
    
    She / quit / **working for the company**.
    
29. 영어를 마스터하는 것은 / 쉽지 않다 (주어)
    
    **Mastering English** / is not / easy.
    
30. 그 도둑은 / 인정했다 / 돈을 훔친 것을 (목적어)
    
    The thief / admitted / **stealing the money**.

### 3. 마무리 포인트 (Exam Tips)

1. **동명사만 좋아하는 동사들 (암기 필수!)**
    - `Enjoy`, `Finish`, `Stop`, `Mind`, `Give up`, `Avoid`, `Practice`, `Quit`, `Put off`, `Suggest`
    - *팁: 앞 글자를 따서 외우기도 합니다 (MEGA PASS 등).*
2. **전치사 뒤에는 무조건 -ing**
    - `of`, `in`, `at`, `about`, `for`, `with`, `without` 뒤에 동사가 오면 100% 동명사입니다.
3. **함정 카드: Look forward to -ing**
    - 여기서 `to`는 투부정사가 아니라 전치사입니다. 그래서 뒤에 동사원형이 오면 **오답**입니다! (`Look forward to see` (X) -> `Look forward to seeing` (O))
