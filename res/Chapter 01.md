### Chapter 01. 투부정사의 부사적 용법 (목적/원인/결과)

### 1. 예문 연습 (Sentence Practice)

1. 나는 / 갔다 / 도서관에 / **책을 빌리기 위해**
    
    I / went / to the library / **to borrow a book**.
    
2. 그녀는 / 공부한다 / 열심히 / **의사가 되기 위해**
    
    She / studies / hard / **to be a doctor**.
    
3. 우리는 / 모였다 / 함께 / **그 노래를 연습하기 위해**
    
    We / gathered / together / **to practice the song**.
    
4. 그는 / 켰다 / 불을 / **그의 열쇠를 찾기 위해**
    
    He / turned on / the light / **to find his key**.
    
5. 나는 / 전화했다 / 그에게 / **질문을 하나 하기 위해**
    
    I / called / him / **to ask a question**.
    
6. 그들은 / 달렸다 / 빨리 / **버스를 잡기 위해**
    
    They / ran / fast / **to catch the bus**.
    
7. 그녀는 / 샀다 / 밀가루를 / **쿠키를 만들기 위해**
    
    She / bought / flour / **to make cookies**.
    
8. 나는 / 모으고 있다 / 돈을 / **유럽으로 여행 가기 위해**
    
    I / am saving / money / **to travel to Europe**.
    
9. 그는 / 갔다 / 공원에 / **그의 친구들을 만나기 위해**
    
    He / went / to the park / **to meet his friends**.
    
10. 눌러주세요 / 이 버튼을 / **기계를 시작하기 위해**
    
    Please press / this button / **to start the machine**.
    
11. 나는 / 멈췄다 / 길에서 / **사진을 한 장 찍기 위해**
    
    I / stopped / on the street / **to take a photo**.
    
12. 지호는 / 일어났다 / 일찍 / **학교에 늦지 않기 위해**
    
    Jiho / got up / early / **not to be late for school**.
    
13. 우리는 / 사용해야 한다 / 더 적은 플라스틱을 / **지구를 구하기 위해**
    
    We / should use / less plastic / **to save the earth**.
    
14. 그녀는 / 방문했다 / 런던을 / **영어를 배우기 위해**
    
    She / visited / London / **to learn English**.
    
15. 나는 / 기쁘다 / **너를 다시 만나서** (원인)
    
    I / am glad / **to meet you again**.
    
16. 그들은 / 놀랐다 / **그 뉴스를 듣고서** (원인)
    
    They / were surprised / **to hear the news**.
    
17. 그녀는 / 정말 행복했다 / **그 상을 받아서** (원인)
    
    She / was really happy / **to win the prize**.
    
18. 나는 / 유감이다 / **그것에 대해 듣게 되어** (원인)
    
    I / am sorry / **to hear that**.
    
19. 그는 / 자랐다 / **(그래서) 유명한 과학자가 되었다** (결과)
    
    He / grew up / **to be a famous scientist**.
    
20. 그 소년은 / 자랐다 / **(그래서) 훌륭한 축구 선수가 되었다** (결과)
    
    The boy / grew up / **to be a great soccer player**.
    
21. 우리 할머니는 / 사셨다 / **(그래서) 90세가 되셨다** (결과)
    
    My grandmother / lived / **to be ninety**.
    
22. 나는 / 일어났다 / 아침에 / **(그래서) 비가 오는 것을 발견했다** (결과)
    
    I / woke up / in the morning / **to find it raining**.
    
23. 너는 / 필요하다 / 여권이 / **해외로 가기 위해**
    
    You / need / a passport / **to go abroad**.
    
24. 그는 / 썼다 / 안경을 / **칠판을 보기 위해**
    
    He / wore / glasses / **to see the blackboard**.
    
25. 우리는 / 갔다 / 식당에 / **저녁을 먹기 위해**
    
    We / went / to the restaurant / **to have dinner**.
    
26. 톰은 / 켰다 / 컴퓨터를 / **그의 숙제를 하기 위해**
    
    Tom / turned on / the computer / **to do his homework**.
    
27. 그녀는 / 갔다 / 시장에 / **신선한 야채를 사기 위해**
    
    She / went / to the market / **to buy fresh vegetables**.
    
28. 나는 / 갔다 / 나의 방으로 / **음악을 듣기 위해**
    
    I / went / to my room / **to listen to music**.
    
29. 그들은 / 일했다 / 열심히 / **그 프로젝트를 끝내기 위해**
    
    They / worked / hard / **to finish the project**.
    
30. 운동해라 / 규칙적으로 / **건강을 유지하기 위해**
    
    Exercise / regularly / **to stay healthy**.
