# Chapter 05

이번 챕터는 한국어에는 없는 개념이라 학생들이 가장 어려워하는 **현재완료 시제**입니다. 4가지 용법(완료, 경험, 계속, 결과)을 골고루 섞어 예문을 구성했습니다.

---

### Chapter 05. 현재완료 시제 (Present Perfect)

### 0. 핵심 요약 (Key Points)

- **형태:** have (has) + p.p. (과거분사)
- **의미:** 과거에 일어난 일이 현재까지 영향을 미치고 있음을 나타냄.
- **4가지 용법:**
    1. **완료:** 막 ~했다 (just, already, yet)
    2. **경험:** ~해본 적이 있다 (ever, never, before, once)
    3. **계속:** (과거부터) 쭉 ~해오고 있다 (for, since)
    4. **결과:** ~해버렸다 (그래서 지금은 없다) (lost, gone)

### 1. 예문 연습 (Sentence Practice)

1. 나는 / 잃어버렸다 / 내 열쇠를 (그래서 지금 없다)
    
    I / **have lost** / my key.
    
2. 그녀는 / 살아왔다 / 여기에서 / 10년 동안
    
    She / **has lived** / here / for 10 years.
    
3. 너는 / 먹어본 적 있니 / 김치를?
    
    **Have you ever eaten** / Kimchi?
    
4. 나는 / 막 끝냈다 / 내 숙제를
    
    I / **have just finished** / my homework.
    
5. 그는 / 가버렸다 / 미국으로 (그래서 지금 여기 없다)
    
    He / **has gone** / to America.
    
6. 우리는 / 알고 지냈다 / 서로를 / 2010년부터
    
    We / **have known** / each other / since 2010.
    
7. 나는 / 본 적이 없다 / 호랑이를
    
    I / **have never seen** / a tiger.
    
8. 그 기차는 / 이미 떠났다
    
    The train / **has already left**.
    
9. 너는 / 끝냈니 / 그 보고서를 / 벌써(아직)?
    
    **Have you finished** / the report / yet?
    
10. 그녀는 / 샀다 / 새 차를 (그래서 지금 새 차가 있다)
    
    She / **has bought** / a new car.
    
11. 나는 / 방문한 적이 있다 / 제주도를 / 두 번
    
    I / **have visited** / Jeju Island / twice.
    
12. 비가 왔다 / 3일 동안 (계속)
    
    It / **has rained** / for three days.
    
13. 톰은 / 부러졌다 / 그의 다리가 (그래서 지금 다친 상태다)
    
    Tom / **has broken** / his leg.
    
14. 나는 / 만난 적이 없다 / 그를 / 전에
    
    I / **haven't met** / him / before.
    
15. 우리 아빠는 / 끊었다 / 담배를 (그래서 지금 안 피운다)
    
    My father / **has quit** / smoking.
    
16. 그들은 / 결혼한 상태다 / 20년 동안
    
    They / **have been married** / for 20 years.
    
17. 우리는 / 막 도착했다 / 역에
    
    We / **have just arrived** / at the station.
    
18. 그녀는 / 잃어버렸다 / 그녀의 지갑을
    
    She / **has lost** / her wallet.
    
19. 너는 / 타본 적 있니 / 말을?
    
    **Have you ever ridden** / a horse?
    
20. 나는 / 쳐왔다 / 피아노를 / 내가 7살 때부터
    
    I / **have played** / the piano / since I was seven.
    
21. 그는 / 돌아오지 않았다 / 아직
    
    He / **hasn't come** / back / yet.
    
22. 누군가가 / 훔쳐 갔다 / 내 자전거를
    
    Someone / **has stolen** / my bike.
    
23. 나는 / 읽어봤다 / 저 책을 / 세 번
    
    I / **have read** / that book / three times.
    
24. 그녀는 / 아팠다 / 지난주부터 (지금도 아프다)
    
    She / **has been** / sick / since last week.
    
25. 나는 / 이미 청소했다 / 내 방을
    
    I / **have already cleaned** / my room.
    
26. 너는 / 들어본 적 있니 / 이 노래에 대해?
    
    **Have you ever heard** / of this song?
    
27. 나의 삼촌은 / 가버렸다 / 파리로 (지금 여기 없음)
    
    My uncle / **has gone** / to Paris.
    
28. 우리는 / 공부해왔다 / 영어를 / 5년 동안
    
    We / **have studied** / English / for 5 years.
    
29. 나는 / 결정하지 못했다 / 아직
    
    I / **haven't decided** / yet.
    
30. 그녀는 / 잊어버렸다 / 그의 이름을 (그래서 지금 모른다)
    
    She / **has forgotten** / his name.

### 3. 마무리 포인트 (Exam Tips)

1. **명백한 과거 부사와는 절대 함께 쓸 수 없음!**
    - `yesterday`, `last night`, `in 2010`, `ago`, `when`과 같은 단어가 보이면 무조건 **과거 시제**를 써야 합니다.
    - (X) I have lost my bag **yesterday**.
    - (O) I **lost** my bag **yesterday**.
2. **가버렸다(gone) vs 가본 적 있다(been)**
    - `She has gone to Paris.` (그녀는 파리로 가버렸다 -> **지금 여기 없다**)
    - `She has been to Paris.` (그녀는 파리에 가본 적이 있다 -> **경험**)
    - *주의: "I have gone to..."나 "We have gone to..."는 "나/우리가 여기 없는데 말하는 것"이 되므로 논리적으로 거의 쓰지 않습니다.*
3. **For vs Since**
    - `For` + **기간** (숫자): for 3 years, for a week.
    - `Since` + **시작 시점**: since yesterday, since 2010, since I was young.
