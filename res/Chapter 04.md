# Chapter 04

이번 챕터는 시험에서 **오답률이 상당히 높은** 파트입니다.

학생들이 가장 많이 헷갈려 하는 것이 **"ing(현재분사)"**를 쓸지, **"p.p(과거분사)"**를 쓸지 고르는 문제입니다. 특히 **감정 동사**(excite, bore, tire 등)를 형용사로 바꿀 때 누가 느끼는 것이고 누가 감정을 주는 것인지 명확히 구분해야 합니다.

---

### Chapter 04. 현재분사와 과거분사 (형용사 역할)

### 0. 핵심 요약 (Key Points)

- **현재분사 (V-ing):** 능동(~하는) / 진행(~하고 있는) / **감정을 유발하는**
    - *Example:* sleeping baby (잠자고 있는 아기), exciting game (신나는 게임)
- **과거분사 (V-ed/p.p):** 수동(~된, ~해진) / 완료(~한) / **감정을 느끼는**
    - *Example:* broken window (깨진 창문), excited fans (신난 팬들)
- **위치:** 명사 앞에서 꾸미거나, 길어지면 명사 뒤에서 꾸며줍니다.

---

### 1. 예문 연습 (Sentence Practice)

1. 봐라 / 저 울고 있는 아기를
    
    Look at / **the crying baby**.
    
2. 조심해라 / 그 깨진 유리를
    
    Watch out for / **the broken glass**.
    
3. 그 잠자고 있는 개는 / 이다 / 귀여운
    
    **The sleeping dog** / is / cute.
    
4. 나는 / 좋아한다 / 삶은 달걀을
    
    I / like / **boiled eggs**.
    
5. 그 게임은 / 이었다 / 정말 흥미진진한
    
    The game / was / **really exciting**.
    
6. 그 관중들은 / 이었다 / 매우 흥분한
    
    The spectators / were / **very excited**.
    
7. 그 소년은 / 축구를 하고 있는 / 이다 / 내 남동생
    
    The boy / **playing soccer** / is / my brother.
    
8. 그 책은 / 영어로 쓰여진 / 이다 / 어려운
    
    The book / **written in English** / is / difficult.
    
9. 이것은 / 이다 / 지루한 영화
    
    This / is / **a boring movie**.
    
10. 그 학생들은 / 지루해했다 / 수업 중에
    
    The students / **were bored** / during the class.
    
11. 너는 아니 / 저 소녀를 / 문 옆에 서 있는?
    
    Do you know / the girl / **standing by the door**?
    
12. 나는 / 받았다 / 편지 한 통을 / 제니에 의해 쓰여진
    
    I / received / a letter / **written by Jenny**.
    
13. 짖는 개는 / 좀처럼 물지 않는다
    
    **A barking dog** / seldom / bites.
    
14. 그들은 / 발견했다 / 그 숨겨진 보물을
    
    They / found / **the hidden treasure**.
    
15. 그 뉴스는 / 이었다 / 충격적인
    
    The news / was / **shocking**.
    
16. 우리는 / 충격을 받았다 / 그 뉴스에
    
    We / **were shocked** / at the news.
    
17. 그 남자는 / 신문을 읽고 있는 / 이다 / 우리 선생님
    
    The man / **reading a newspaper** / is / my teacher.
    
18. 그 가방은 / 책으로 채워진 / 이다 / 무거운
    
    The bag / **filled with books** / is / heavy.
    
19. 그녀는 / 가지고 있다 / 매력적인 미소를
    
    She / has / **a charming smile**.
    
20. 나는 / 샀다 / 중고 차를
    
    I / bought / **a used car**.
    
21. 그 결과는 / 이었다 / 실망스러운
    
    The result / was / **disappointing**.
    
22. 나는 / 실망했다 / 그 결과에
    
    I / **was disappointed** / with the result.
    
23. 그 여자는 / 커피를 마시고 있는 / 이다 / 나의 이모
    
    The lady / **drinking coffee** / is / my aunt.
    
24. 모든 사람들은 / 파티에 초대된 / 가졌다 / 즐거운 시간을
    
    Everyone / **invited to the party** / had / a good time.
    
25. 봐라 / 그 떨어지는 잎들을
    
    Look at / **the falling leaves**.
    
26. 그 떨어진 잎들은 (낙엽) / 이다 / 갈색
    
    **The fallen leaves** / are / brown.
    
27. 나는 / 좋아하지 않는다 / 튀긴 음식을
    
    I / don't like / **fried food**.
    
28. 내 직업은 / 이다 / 피곤하게 하는 (힘든)
    
    My job / is / **tiring**.
    
29. 나는 / 이다 / 매우 피곤한
    
    I / am / **very tired**.
    
30. 경찰은 / 찾았다 / 그 도난당한 돈을
    
    The police / found / **the stolen money**.

### 3. 마무리 포인트 (Exam Tips)

1. **감정 형용사 구분법 (가장 중요!)**
    - 주어가 **감정을 주는 원인**(주로 사물/상황)이면 → **ing** (ex. The movie is bor**ing**.)
    - 주어가 **감정을 느끼는 주체**(주로 사람)이면 → **ed** (ex. I am bor**ed**.)
2. **명사 뒤에서 꾸밀 때 (후치 수식)**
    - `The boy playing soccer` 처럼 분사 뒤에 다른 단어(목적어, 전치사구 등)가 붙어서 길어지면 **명사 뒤**로 갑니다.
    - 이때 명사와 분사 사이에는 **주격 관계대명사+be동사** (who is, which is)가 생략된 것으로 봅니다.
3. **자동사의 pp는 '수동'이 아니라 '완료'**
    - `Fallen leaves`는 '떨어지게 된 잎'이 아니라 '이미 다 떨어진 잎(낙엽)'을 의미합니다.
