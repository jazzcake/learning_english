# Chapter 06

이번 챕터는 **과거완료 시제(had + p.p.)**입니다. "과거보다 더 이전의 과거(대과거)"를 나타낼 때 쓰이며, 어떤 일이 먼저 일어났는지 **순서**를 따지는 것이 핵심입니다.

---

### Chapter 06. 과거완료 시제 (Past Perfect)

### 0. 핵심 요약 (Key Points)

- **형태:** had + p.p. (과거분사)
- **의미:** 특정한 과거 시점보다 **더 이전에** 일어난 일 (대과거).
- **주요 쓰임:** 과거의 두 사건 중 먼저 일어난 쪽에 `had p.p.`를 씁니다.
    - *(예: 내가 역에 도착했을 때(과거), 기차는 이미 떠났다(더 과거 -> 대과거))*

---

### 1. 예문 연습 (Sentence Practice)

1. 나는 / 잃어버렸다 / 그 시계를 / 아빠가 나에게 주셨던 (대과거)
    
    I / lost / the watch / that my dad **had given** to me.
    
2. 내가 / 도착했을 때 / 역에 / 기차는 이미 떠났다 (대과거)
    
    When / I / arrived / at the station, / the train **had already left**.
    
3. 그녀는 / 본 적이 없었다 / 사자를 / 그녀가 동물원에 가기 전에 (경험)
    
    She **had never seen** / a lion / before she went to the zoo.
    
4. 그는 / 말했다 / 그가 끝냈다고 / 그 일을 (대과거)
    
    He / said / that he **had finished** / the work.
    
5. 나는 / 알아봤다 / 그를 / 왜냐하면 내가 만난 적이 있기 때문에 / 그를 전에 (원인-대과거)
    
    I / recognized / him / because I **had met** / him before.
    
6. 그 도로는 / 젖어 있었다 / 왜냐하면 비가 왔었기 때문에 / 밤새도록
    
    The road / was wet / because it **had rained** / all night.
    
7. 그녀는 / 살아왔었다 / LA에서 / 5년 동안 / 내가 그곳으로 이사했을 때 (계속)
    
    She **had lived** / in LA / for 5 years / when I moved there.
    
8. 나는 / 먹을 수 없었다 / 점심을 / 왜냐하면 내가 이미 먹었기 때문에 (완료)
    
    I / couldn't eat / lunch / because I **had already eaten**.
    
9. 그 영화는 / 이미 시작했다 / 우리가 영화관에 도착했을 때
    
    The movie **had already started** / when we got to the theater.
    
10. 그는 / 깨달았다 / 그가 두고 온 것을 / 그의 지갑을 / 집에
    
    He / realized / that he **had left** / his wallet / at home.
    
11. 나는 / 찾을 수 없었다 / 그 우산을 / 내가 어제 샀던
    
    I / couldn't find / the umbrella / which I **had bought** / yesterday.
    
12. 그녀는 / 매우 피곤했다 / 왜냐하면 그녀가 일했기 때문에 / 하루 종일
    
    She / was very tired / because she **had worked** / all day.
    
13. 그는 / 아팠었다 / 일주일 동안 / 그가 의사에게 갔을 때
    
    He **had been** / ill / for a week / when he went to the doctor.
    
14. 나는 / 물었다 / 그녀에게 / 어디에 그녀가 있었는지
    
    I / asked / her / where she **had been**.
    
15. 선생님은 / 물어보셨다 / 내가 했는지 / 내 숙제를
    
    The teacher / asked / if I **had done** / my homework.
    
16. 그들은 / 막 끝마쳤다 / 저녁 식사를 / 초인종이 울렸을 때
    
    They **had just finished** / dinner / when the bell rang.
    
17. 나는 / 원했다 / 그 영화를 보기를 / 왜냐하면 내가 들었기 때문에 / 좋은 평을
    
    I / wanted / to see the movie / because I **had heard** / good things.
    
18. 톰은 / 타본 적이 없었다 / 비행기를 / 그 여행 전에
    
    Tom **had never flown** / in an airplane / before that trip.
    
19. 그녀는 / 나에게 말했다 / 그녀가 잃어버렸다고 / 그녀의 휴대전화를
    
    She / told me / that she **had lost** / her phone.
    
20. 나는 / 발견했다 / 누군가가 깼다는 것을 / 그 창문을
    
    I / discovered / that someone **had broken** / the window.
    
21. 그 경기는 / 끝났다 / 내가 TV를 켰을 때쯤에
    
    The game **had ended** / by the time / I turned on the TV.
    
22. 그녀는 / 공부해왔었다 / 영어를 / 3년 동안 / 그녀가 미국에 가기 전에
    
    She **had studied** / English / for 3 years / before she went to the US.
    
23. 그는 / 행복함을 느꼈다 / 왜냐하면 그가 통과했기 때문에 / 그 시험을
    
    He / felt happy / because he **had passed** / the exam.
    
24. 나는 / 열 수 없었다 / 그 문을 / 왜냐하면 내가 잃어버렸기 때문에 / 열쇠를
    
    I / couldn't open / the door / because I **had lost** / the key.
    
25. 그것은 / 이었다 / 최고의 피자 / 내가 먹어본 적이 있었던
    
    It / was / the best pizza / I **had ever eaten**.
    
26. 수지는 / 요리했다 / 그 요리를 / 그녀가 엄마에게 배웠던
    
    Suji / cooked / the dish / which she **had learned** / from her mom.
    
27. 그는 / 보여주었다 / 나에게 / 사진들을 / 그가 파리에서 찍었던
    
    He / showed / me / the photos / he **had taken** / in Paris.
    
28. 눈이 / 그쳤다 / 우리가 밖으로 나갔을 때
    
    The snow **had stopped** / when we went out.
    
29. 그녀는 / 몰랐다 / 그가 떠났다는 것을
    
    She / didn't know / that he **had left**.
    
30. 우리는 / 배가 고팠다 / 왜냐하면 우리가 먹지 않았기 때문에 / 아무것도
    
    We / were hungry / because we **hadn't eaten** / anything.

### 3. 마무리 포인트 (Exam Tips)

1. **시간의 순서 찾기 (타임라인 그리기)**
    - 시험 문제에서는 두 사건을 주고 동사의 형태를 알맞게 쓰라는 문제가 나옵니다.
    - "내가 도착했다(과거)" vs "기차가 떠났다(더 과거)" -> 기차가 먼저 떠났으므로 **had left**를 써야 합니다.
2. **접속사 When / Before / After**
    - `When`이나 `Before`를 쓸 때는 앞뒤 문맥을 보고 무엇이 먼저인지 판단해야 합니다.
    - `After`가 쓰인 경우: `After he had finished his work, he went home.` (일 끝낸 게 먼저)
3. **현재완료 vs 과거완료**
    - 기준 시점이 **'현재'**면 현재완료 (지금까지 ~해왔다).
    - 기준 시점이 **'과거'**면 과거완료 (그때까지 ~했었다).
