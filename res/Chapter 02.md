# Chapter 02

이번 챕터는 **명사적 용법**(~하는 것)과 **형용사적 용법**(~할, ~하는)을 다룹니다. 특히 형용사적 용법 뒤에 전치사가 붙는 경우(예: live in, talk to)는 **시험에 매우 자주 나오는 포인트**이니 예문에 꼭 포함시켰습니다.

---

### Chapter 02. 투부정사의 명사적/형용사적 용법

### 1. 예문 연습 (Sentence Practice)

1. 나는 / 원한다 / 훌륭한 수의사가 되는 것을 (명사-목적어)
    
    I / want / **to be a good vet**.
    
2. 내 취미는 / 이다 / 우표를 수집하는 것 (명사-보어)
    
    My hobby / is / **to collect stamps**.
    
3. 그는 / 결정했다 / 해외로 가기로 (명사-목적어)
    
    He / decided / **to go abroad**.
    
4. 제니는 / 가지고 있다 / 마실 무언가를 (형용사)
    
    Jenny / has / **something to drink**.
    
5. 지금은 / 이다 / 잠자리에 들 시간 (형용사)
    
    It / is / **time to go to bed**.
    
6. 우리는 / 계획했다 / 박물관을 방문하는 것을 (명사-목적어)
    
    We / planned / **to visit the museum**.
    
7. 나는 / 가지고 있다 / 끝내야 할 많은 숙제를 (형용사)
    
    I / have / **a lot of homework to finish**.
    
8. 그녀는 / 약속했다 / 거짓말하지 않기로 (명사-목적어)
    
    She / promised / **not to tell a lie**.
    
9. 나의 꿈은 / 이다 / 세계를 여행하는 것 (명사-보어)
    
    My dream / is / **to travel the world**.
    
10. 너는 / 가지고 있니 / 읽을 책을? (형용사)
    
    Do you / have / **a book to read**?
    
11. 그는 / 필요하다 / 앉을 의자가 (형용사-전치사 주의)
    
    He / needs / **a chair to sit on**.
    
12. 외국어를 배우는 것은 / 이다 / 어렵지 않다 (명사-주어)
    
    **To learn a foreign language** / is / not difficult.
    
13. 나는 / 희망한다 / 너를 곧 보기를 (명사-목적어)
    
    I / hope / **to see you soon**.
    
14. 그들은 / 찾고 있다 / 살 집을 (형용사-전치사 주의)
    
    They / are looking for / **a house to live in**.
    
15. 톰은 / 원하지 않는다 / 혼자 있는 것을 (명사-목적어)
    
    Tom / doesn't want / **to be alone**.
    
16. 우리에게 주세요 / 차가운 마실 것을 (형용사)
    
    Give us / **something cold to drink**.
    
17. 그녀는 / 동의했다 / 우리를 돕기로 (명사-목적어)
    
    She / agreed / **to help us**.
    
18. 나는 / 필요하다 / 가지고 쓸 펜이 (형용사-전치사 주의)
    
    I / need / **a pen to write with**.
    
19. 너는 / 원하니 / 산책하러 가는 것을? (명사-목적어)
    
    Do you / want / **to go for a walk**?
    
20. 이것은 / 이다 / 그 문제를 해결할 최선의 방법 (형용사)
    
    This / is / **the best way to solve the problem**.
    
21. 나는 / 가지고 있지 않다 / 입을 옷을 (형용사)
    
    I / don't have / **clothes to wear**.
    
22. 그의 직업은 / 이다 / 아이들을 가르치는 것 (명사-보어)
    
    His job / is / **to teach children**.
    
23. 그 노부인은 / 필요하다 / 이야기할 누군가가 (형용사-전치사 주의)
    
    The old lady / needs / **someone to talk to**.
    
24. 우리는 / 기대한다 / 그 경기에서 이기기를 (명사-목적어)
    
    We / expect / **to win the game**.
    
25. 있다 / 많은 장소들이 / 서울에는 방문할 (형용사)
    
    There are / many places / **to visit in Seoul**.
    
26. 시작하는 것은 / 이다 / 중요하다 (명사-주어)
    
    **To start** / is / important.
    
27. 나는 / 가지고 있다 / 쓸 종이가 (형용사-전치사 주의)
    
    I / have / **paper to write on**.
    
28. 그녀는 / 거절했다 / 나의 제안을 받아들이는 것을 (명사-목적어)
    
    She / refused / **to accept my offer**.
    
29. 나는 / 필요하다 / 약간의 돈이 / 쓸 (형용사)
    
    I / need / **some money to spend**.
    
30. 그들은 / 원한다 / 유명해지는 것을 (명사-목적어)
    
    They / want / **to be famous**.
